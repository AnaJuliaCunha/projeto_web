Git e GitHub

O que é o Git?

Git é usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software. Ele permite mais flexibilidade no fluxo de trabalho, segurança e desempenho.Em resumo, ele é uma ferramenta de controle de versão local

O que é o GitHub?

GitHub é uma plataforma usada para gerenciar, hospedar códigos e arquivos, além de facilitar a criação de ambientes de colaboração entre pessoas desenvolvedoras. O GitHub usa como sistema de controle de versão o Git. Ou seja, o GitHub facilita e auxilia a usar o Git no dia a dia.

Como configurar sua conta no Git Bash, criando e clonando um repositório:

  • Primeiramente é necessário ter baixado o Git na sua máquina
  • Após abrir o Git Bash, digite git --version para identificar a versão do Git Bash instalado.
  • Digite git config --global user.name "seu nome" para configurar nome do usuário
  • Digite git config --global user.email "seu@email" para configurar e-mail do usuário
  • Abrir a pasta de destino digite cd Documents/(ou a pasta de destino)
  • Digite ls para listar os itens da pasta
  • Para criar/abrir uma pasta, digite mkfir nomeDaPasta
  • Digite cd nomeDaPasta para abrir a pasta desejada
  • Digite git init para configuração inicial de um novo repositório. A execução desse comando cria um novo subdiretório . git no diretório de trabalho atual. Essa ação também vai criar uma ramificação principal.
  • Para abrir um repositório já existente utilize o comando git clone "link HTTPS/SHH"
  • Utilize code . para abrir a pasta no VScode

Como configurar Chave SSH:

Segue instruções de configuração:

  • Primeiro - Deverá abrir seu Git Bash e verificar se já há uma chave SSH com o comando ls -al ~/.ssh
  • Segundo - Ou crie uma nova chave com o comando ssh-keygen -t ed25519 -C "email@exemplo.com"
  • Terceiro - Inicialize um agente com o comando eval "$(ssh-agent -s)"
  • Quarto - Insira a chave SSH com a linha de código ssh-add ~/.ssh/id_ed25519
  • Quinto - Utilize o comando para copiar para a área de transferência, sua chave SSH clip < ~/.ssh/id_ed25519.pub
  • Sexto - Entre na conta do GitHub e abra suas settings > SSH and GPG keys
  • Sétimo - Vá em adicionar uma chave SSH, e dê um nome a essa chave para identificação. Depois, cole o código copiado na caixa de texto grande.
  • Oitavo - Use ssh -T git@github.com no bash para conectar.
